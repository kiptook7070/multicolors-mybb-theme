How to install this theme

MYBB Premium Multicolored v1.02

Upload all files from upload folder to your FTP. Overwrite all files if some exists.

1. Go to your main MyBB folder in FTP.

Upload there all files from 'upload' folder.

If some files already exists - overwrite it!

2. Go to ACP and follow these steps

- Select TEMPLATE & STYLES TAB
- Click on in button Import a Theme
- Select THEME NAME .XML from theme folder
- Select check button IGNORE VERSION COMPATIBILITY
- Click IMPORT THEME BUTTON to save and finish uploading process

* - Select Theme as DEFAULT to apply changes.
This theme support all usual plugins from mybb.

3. Go to index page to see the changes.

- Refresh page and enjoy!
